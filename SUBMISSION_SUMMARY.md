## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/Reddy076/Weather.git

# Navigate to project
cd Weather

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
